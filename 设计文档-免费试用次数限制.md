# 免费试用次数限制 — 设计方案

> 创建日期：2026-05-13
> 状态：设计完成，待实现
> 目标：让用户免费试用引流，又不让开发者被无限消耗额度

---

## 使用场景

```
        服务器 Key（开发者）         用户自己的 Key
        ────────────────          ──────────────
使用者    开发者自己                  普通用户（3 次后）
存储      环境变量                    用户浏览器 localStorage
费用      开发者出                    用户自己出（火山引擎免费额度）
次数      无限                       不限（他自己的额度）
安全      Render 环境变量            不离开浏览器，不存服务器
```

---

## 用户流程

```
新用户首次访问
    │ localStorage: saifu_usage = 0
    ▼ 点「开始匹配」
后端用服务器 Key → 成功返回（saifu_usage → 1）
    │
    ├── 第 1-3 次：免费用，不弹窗
    │
    └── 第 4 次 → 弹窗提示
```

## 弹窗设计

### 内容要点

弹窗需包含以下信息层级：

1. **免费次数已用完**（醒目提示）
2. **两个推荐的 API 平台供用户选择**（不为任一平台背书，让用户自己选）
3. **模型选择建议**：越强的模型调研质量越好，但也都免费/极便宜
4. **三条信任保障**（底部，简洁）

### 模型选择提示

弹窗中给用户展示模型档位参考：

| 档位 | 推荐模型 | 调研质量 | 价格 | 适合 |
|------|----------|----------|------|------|
| 旗舰 | DeepSeek v4-pro / 豆包 Seed 2.0 Pro | 最强 | ~5 分/次 | 重要比赛 |
| 均衡 | DeepSeek v4-flash | 强 | ~1.6 分/次 | 日常调研 |
| 省钱 | 豆包 Seed 2.0 Lite | 够用 | ~1.5 分/次 | 大量匹配 |

> 💡 提示文案：「模型越强，竞赛匹配和调研结果越精准。下面两个平台都有免费额度，新用户足够用很久。」

### 两个平台注册入口

| 平台 | 注册地址 | 免费额度 |
|------|----------|----------|
| 火山引擎（豆包） | [console.volcengine.com](https://console.volcengine.com) | 新用户送 50 万 tokens |
| DeepSeek | [platform.deepseek.com](https://platform.deepseek.com) | 新用户送 1000 万 tokens |

### 三条信任保障

> - 你的 Key **只存在你的浏览器里**，我们服务器收不到
> - 每次调用**即用即弃**，不记录、不存盘、不上传 GitHub
> - 你自己去平台注册，**你自己的账号你自己控制**

## 安全设计：密钥不落地

| 防护层 | 做法 |
|---|---|
| 后端 | `user_data.pop("user_api_key", None)` — 存盘前删除 |
| 后端 | 日志不打印 API Key |
| 仓库 | `.gitignore` 加 `backend/data/` |
| 传输 | HTTPS 加密 |
| 存储 | 用户浏览器 localStorage（永不离开浏览器） |

---

## 改动清单（3 文件 + 完全解耦）

| 文件 | 改什么 | 几行 | 影响现有逻辑 |
|---|---|---|---|
| `ai_client.py` | `call_deepseek()` 加 `api_key=None` 参数 | 5 行 | 否 |
| `config.py` | 环境变量改名通用化 | 3 行 | 一次性 |
| `main.py` | 收 `user_api_key` + 存盘前删除 | 8 行 | 否 |
| `app.js` | 计数器 + 弹窗逻辑（独立模块） | 50 行 | 否 |
| `index.html` | 弹窗 HTML + CSS | 30 行 | 纯新增 |
| `.gitignore` | 加 `backend/data/` | 1 行 | 纯安全 |

## 如何删除整个功能

```
入口检查（app.js 前 10 行）
    │
    ├── 想删掉整个功能 → 删这 10 行 + modal HTML → 恢复原样
    ├── 想换付费模式 → 改弹窗文案
    └── 想改次数 → 改 const FREE_LIMIT = 3
```

## 关键常量和锚点

- `const FREE_LIMIT = 3`（app.js）

- `user_api_key` 请求可选字段（main.py）
- `api_key=None` 参数（ai_client.py）
