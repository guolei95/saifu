# 配图Brief模板

> 用法：复制这份模板，每人给自己负责的每张图各填一份，填完直接发给设计队友。不经过小雷转达。
> 设计队友按brief直接画，画完小雷最终过一眼。

---

## 配图Brief：*[图名]*

**所属章节**：第X章 [章节名]
**图类型**：[架构图 / 流程图 / 时间轴 / 数据图 / 概念图]
**尺寸**：[横版 / 竖版，占页面宽度 X%]

**图中要画什么**（分层/分步骤描述）：

- 第1层/步骤1：*[描述要展示什么内容]*
- 第2层/步骤2：*[描述]*
- 第3层/步骤3：*[描述]*
- ...

**文字/数据来源**：对应章节第X.X节，设计队友可直接引用原文
**颜色要求**：主色 #2563EB（项目品牌蓝），辅色按需要
**参考图**：[贴一张你觉得类似的网络截图，或手画草图拍照贴在下面]
**截止时间**：2026年__月__日

## 示例（已填好的参考）

**所属章节**：第4章 产品设计与功能
**图类型**：架构图
**尺寸**：横版，占页面宽度 80%

**图中要画什么**：

- 第1层 用户入口：左侧"Web应用"，右侧"微信小程序"
- 第2层 三大方向：「大创/互联网+」→「商赛」→「数学建模」
- 第3层 五大功能：选赛推荐 → 调研辅助 → 方案生成 → PPT路演 → 赛后沉淀
- 第4层 AI引擎：大语言模型 + RAG + 提示词体系
- 第5层 知识库：赛事库300+ | 模板库 | 案例库

**文字/数据来源**：第4章4.1节产品架构总览
**颜色要求**：主色 #2563EB，各层用蓝系深浅区分
