```

                         窄 ←——————→ 宽

          高   │  保研岛AI竞赛        │  ★ 赛赋
              │  后保研智能           │    （全品类+全流程+AI匹配）
个            │  高顿去保研           │
```

| **右上**（宽×高个性化） | 全品类+智能匹配 = **赛赋目标位置** | 目前空白 | 机会区 |
| **左下**（窄×低个性化） | 单赛事工具，无匹配能力 | 摩课云、InsCode | 低 |
| **右下**（宽×低个性化） | 大而全信息聚合，无AI匹配 | 赛氪、我爱竞赛网 | 高（用户基数大） |

## 四、核心竞争框架：「上限 vs 下限」

### 4.1 一个洞察说透所有竞品的软肋

把全部17个竞品按「帮什么人、解决什么问题」重新排列，呈现出一个清晰的断层：

```
        帮高手提上限
            │
    晨星创投 ████████████  ← 有想法、有材料、有半成品 → 帮你生成成品
    南天信息 ██████
    喵喵OJ  ████
    创赛之星 ████
            │
  ──────────┼──────────  断层线：无人覆盖从0到1
            │
    赛氪     ██          ← 只有信息陈列，不帮你决策
    保研岛   ███         ← 只覆盖100个赛事，数据滞后
    赛赋     ██████████  ← 从「你是谁」开始，带你走完0→1
            │
        帮新手兜下限
```

### 4.2 晨星创投专利的精准解剖

**它是什么**：一个「有材料之后的深度加工系统」
- 多模态解析 → 需要你上传已有文档
- 案例库检索 → 需要你明确竞赛方向
- AI诊断评分 → 需要你有半成品方案
- 材料自动生成 → 基于你提供的内容做优化

**它不是什么**：
- ❌ 不能告诉一个新手「你该参加什么比赛」
- ❌ 不能帮一个零基础学生「从哪开始准备」
- ❌ 不能替一个信息匮乏的双非学生「找到适合的竞赛」

**它的用户画像**：已有明确竞赛方向、已积累部分材料、需要高效打磨的「进阶用户」

### 4.3 赛赋的定位：填平断层线

```
晨星创投：有食材 → 帮你炒出一盘好菜
赛赋：    你饿了 → 告诉你去哪个菜市场、买什么菜、怎么切

晨星让人从 80分 → 95分
赛赋让人从 0分  → 60分
```

**而大学生竞赛市场，卡在0分的人远比卡在80分的人多。**

尤其对于双非院校学生：
- 他们不是「方案写得不够好」，而是「根本不知道有什么比赛能参加」
- 他们不是「材料不够完善」，而是「不知道这个比赛需要什么材料」
- 他们不是「路演讲得不够流畅」，而是「连第一步都没迈出去」

**赛赋的价值不在「上限有多高」，而在「下限兜得住」。**

## 五、战略建议：赛赋应该怎么走

### 5.1 总体策略：错位竞争，不正面刚

```
赛氪/我爱竞赛网 = 信息超市（什么都有，什么都不帮你选）
保研岛/后保研   = 精品店（只卖保研相关的100款）
创赛之星        = 打磨工坊（只帮你改方案，不帮你找比赛）
晨星创投        = 进阶厨房（给有食材的人炒菜）

赛赋 = 入门向导 + 全流程管家
       从「你是谁」出发，带你从0走到1
```

**核心差异一句话**：不是「告诉你有什么比赛」，也不是「帮你把方案写更好」——而是「让一个零基础的人，从不知道自己能参加什么，到拿着方案站上赛场」。

### 5.2 三大战略选择

#### 选择一：先打「全品类AI推荐」这张牌（推荐）

**理由**：这是市场上唯一空白的右上角象限。赛氪有信息没AI，保研岛有AI但只有100个比赛，创赛之星有AI但不做推荐。

**打法**：
- MVP阶段聚焦「选赛推荐」这一个核心功能，做到极致
- 输入：专业、年级、目标（保研/考研/就业/兴趣）、可投入时间
- 输出：Top 5匹配竞赛 + 每项的推荐理由（含加分权重、获奖率、难度评估）
- 暂不做全流程，先验证「AI匹配」这个核心价值

**优势**：单点突破快、用户感知强、容易传播（「测测你适合参加什么竞赛」）

#### 选择二：先打「保研加分精准匹配」这张牌

**理由**：保研岛/后保研虽然也做保研，但它们只覆盖100+竞赛，且数据更新滞后。如果你能建立起全国高校保研加分政策的结构化数据库，就是不可替代的护城河。

**打法**：
- 聚焦双非院校保研加分政策（985/211学生信息渠道多，双非学生最需要帮助）
- 爬取+人工整理各校学院级的加分细则
- 做成「输入你的学校和专业 → 告诉你哪些竞赛能加分、加多少分」

**风险**：数据采集工作量巨大，一人团队可能撑不住。

#### 选择三：先打「双非学生」这张牌（强烈推荐）

**理由**：
- 你之前调研已经验证：双非学生是信息差最大的群体
- 985/211学生有学长传承、学校推送、老师指导，信息渠道通畅
- 赛氪/保研岛的用户以985/211为主
- 双非学生规模更大（全国3000+高校，双非占绝大多数）

**打法**：
- 品牌定位：**「双非学生专属竞赛导航」**
- 功能聚焦：不追求5万+赛事覆盖，先做好1000个对双非学生最友好、门槛最合理、加分最实用的竞赛
- 营销切入点：「不是只有985才能拿奖」

### 5.3 我的建议：选择三 × 选择一 的组合

**第一步（当前→MVP上线）**：打「双非学生 × AI选赛推荐」
- 做一个网页版选赛推荐工具
- 输入你的学校（双非）、专业、年级、目标
- 输出Top 5竞赛推荐 + 详细理由
- 目标：让100个双非学生试用，验证匹配准确率

**第二步（MVP验证后）**：加入「保研加分数据库」
- 重点搜集50所双非院校的竞赛加分政策
- 作为推荐的底层权重
- 这是保研岛没做、赛氪不会做的数据壁垒

**第三步（有用户基础后）**：拓展全流程辅助
- 方案写作辅助 → 答辩模拟 → 赛后复盘
- 逐步从「选赛工具」升级为「竞赛全流程平台」

## 六、风险预警与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 晨星创投专利产品化 | 中 | 高 | 持续关注，他们的专利偏「方案生成」，赛赋差异化在「推荐匹配」 |
| 赛氪推出AI功能 | 中 | 高 | 赛氪商业模式决定它不会重点做个性化推荐（和主办方收费冲突），但仍需关注 |
| 保研岛扩品类 | 低 | 中 | 保研岛品牌即「保研」，扩品类会稀释定位 |
| 巨头入场 | 低 | 极高 | 不可控，但巨头更可能收购而非自建，做好产品就是最好的防御 |
| 一人团队撑不住 | 中 | 极高 | **最现实的风险**。建议尽快找1-2个队友（技术+商科） |

## 八、一句话总结

> **晨星帮高手从80分提到95分，赛赋帮新手从0分走到60分。大学生竞赛市场，卡在0分的人远比卡在80分的人多。赛赋的价值不在上限有多高，而在下限兜得住——让每一个零基础、信息匮乏的双非学生，都能迈出竞赛的第一步。**
