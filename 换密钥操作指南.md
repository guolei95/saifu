# 🔑 换 DeepSeek API Key 正确操作

> ⚠️ 后端跑在 Render 云端，**改本地 .env 完全没用！别碰本地文件！**

## 正确步骤（三步，5 分钟）

| 步骤 | 操作 | 地址 |
|------|------|------|
| 1 | DeepSeek 平台：**先建新密钥 → 再删旧密钥** | https://platform.deepseek.com |
| 2 | Render 环境变量：更新 `DEEPSEEK_API_KEY` | https://dashboard.render.com → saifu-backend → Environment |
| 3 | 验证 | https://guolei95.github.io/saifu/ 试一次匹配 |

## 密钥读取链路

```
Render 环境变量 → 云端 config.py → 云端 ai_client.py → DeepSeek API
                     ↑
        你操作的唯一入口（改 Render，别改本地）
```

## 常见错误

| ❌ 错误 | 为什么没用 |
|--------|-----------|
| 改本地 `.env` 文件 | 后端在 Render 云端，不读你电脑上的文件 |
| 改 `config.py` 硬编码密钥 + push | 密钥直接泄露到 GitHub |
| 只删 DeepSeek 旧密钥，不更新 Render | 网站直接挂（旧密钥失效） |
