# 赛赋项目拿奖分析与差距评估

> 生成日期：2025-05-08
> 数据来源：`01-调研/灵异调研/` 文件夹内11份调研文件（17个竞品分析、AI调研报告、本地资源摸底、面立方案例等）

---

## 一、项目定位：AI驱动的竞赛全流程辅助平台

赛道判断：✅ 赛道选对了，定位足够差异化。但以当前「1人、无合作方、无用户数据」的状态，拿省赛奖需补至少3块板，拿国赛奖差距很大。

---

## 二、拿奖公式

```
拿奖 = 真实需求(要有数据) × 合作方背书(要有章) × 做得出来(要有东西) × 团队完整(要有人)
```

| 要素 | 满分要求 | 当前状态 | 差距 |
|------|---------|---------|------|
| **真实需求** | 100份问卷 + 10人访谈 + 数据可视化 | ❌ 零用户调研 | 🔴 致命 |
| **合作方背书** | 企业合作意向书 / 政府合作函 / 专家推荐信 | ❌ 零合作方 | 🔴 致命 |
| **做得出来** | 网页MVP可演示 + 技术架构图 + 核心流程跑通 | ✅ V1 CLI完成，HTML版PPT已有 | 🟡 MVP未做 |
| **团队完整** | 3-5人，覆盖技术+商科+设计 | ❌ 1人 | 🟡 评审占20分 |

## 三、互联网+大赛评审维度（创意组）

| 维度 | 权重 | 评委关注点 |
|------|:---:|------|
| **教育维度** | 30分 | 是否体现大学生能力成长？是否解决教育/学生真实问题？ |
| **创新维度** | 20分 | 技术创新 + 业务创新。四维匹配+全流程是加分项 |
| **团队维度** | 20分 | 成员结构完整？有无指导老师？ |
| **商业维度** | 20分 | 谁付钱、怎么赚钱、凭什么付钱？ |
| **社会价值** | 10分 | 「双非学生信息公平」可打这张牌 |

## 四、已有优势（调研层面）

1. **竞品分析量级超标**：17个竞品覆盖国内+国外，大多数参赛团队只分析2-3个
2. **核心洞察「断层线」**：帮新手从0到1无人覆盖——答辩杀手级论点
3. **面立方对标精准**：同为双非+软件+AI+学生服务，全国特等奖，证明赛道天花板够高
4. **黄石本地资源摸底扎实**：政策、企业、高校、孵化器全列
5. **多专业批量测试验证**：6个专业均可跑出有效推荐

## 五、各层次要求

### 🥉 校赛入围 / 学院推荐（差距约30%）

- [ ] 完整商业计划书（按互联网+模板）
- [ ] 可演示的网页MVP（Mock数据即可）
- [ ] 指导老师（无指导老师无法报名）

### 🥈 省赛获奖（差距约60%）

校赛要求 +：

- [ ] 100份问卷调研（本校+其他双非学校）
- [ ] 软著受理通知书
- [ ] 1家合作方（黄石企点创 / 学校双创学院）
- [ ] 3-4人完整团队
- [ ] 10人以上用户试用心得

### 🥇 国赛级别（差距约85%）

省赛要求 +：

- [ ] 500+真实用户数据
- [ ] 正式企业/政府合作协议（盖章，非意向书）
- [ ] 1项实用新型专利 + 1-2篇论文
- [ ] 可量化效果数据（推荐准确率X%、参赛转化率提升X%）
- [ ] 商业模型初步验证（有收入）

## 六、当前最优先行动（按优先级）

### 🔴 P0：找指导老师 + 找队友（阻塞项）

- 无指导老师无法报名
- 1人参赛，团队维度直接扣分
- 至少招募：1商科（商业计划书/财务）+ 1设计（PPT/UI）

### 🔴 P0：100份问卷调研

- 当前PPT全是自述和二手数据，无一手数据支撑
- 核心问题：你知道多少竞赛？参加过吗？为什么不参加？最需要什么帮助？
- 工具：问卷星/腾讯问卷

### 🟡 P1：网页MVP + 软著申请

- 评委看不了命令行，需要网页展示
- 单页HTML：输入专业/年级/目标 → 输出Top 5推荐竞赛
- 同步整理代码+架构文档，提交软著

---

## 七、软件平台类项目避坑要点

评审常见偏见：「不就是做个App/网站吗」。三招破局：

1. **技术架构图**：复杂本身就是壁垒
2. **数据说话**：不是讲代码，是讲效果（准确率提升X%、效率提升X倍）
3. **合作方背书**：有企点创/大唐融合/学校双创学院的章，比讲100页技术管用

---

## 八、总结

调研已做到位（17竞品、断层线洞察、面立方对标），但接下来必须从「纸上论证」→「实物+数据+合作关系」。

**最缺的三样：人（队友+导师）、数据（100份问卷）、合作方（一个章）。**
