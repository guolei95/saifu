# 提示词：把调研结果转成 Word / PDF 文档

> 复制以下内容发给另一个 AI，让它帮你实现。

---

## 项目背景

我有一个竞赛匹配网站（`saifu/`），当用户提交画像后，后端会调用 AI 生成一份「个性化竞赛调研报告」。调研结果是 JSON 格式，包含推荐竞赛、备赛建议、风险提示等内容。现在需要把这份调研结果转成一份可以直接发给用户的 **Word 文档** 或 **PDF 文件**。

## 你的任务

在 `saifu/` 项目中新增一个功能：**把调研结果 JSON 一键导出为格式化的 Word（.docx）文档**。

### 1. 后端：新增导出 API

在 `saifu/backend/main.py` 中新增路由：

```
POST /api/export-report
```

接收 body：
```json
{
  "user_name": "张三",
  "user_school": "清华大学",
  "research_result": {
    "recommendations": [
      {
        "name": "中国大学生计算机设计大赛",
        "level": "国家级",
        "deadline": "2026-06-15",
        "reason": "匹配你的 AI/大模型方向，且你有 Python 基础",
        "preparation": "准备一个 AI 项目作品，备好答辩 PPT"
      }
    ],
    "advice": "建议你在 7-8 月集中备赛...",
    "risks": "你的指导老师缺失，建议提前联系..."
  }
}
```

后端要做的事：
1. 使用 `python-docx` 库生成 Word 文档（.docx）
2. 文档排版要求见下方
3. 把生成的 .docx 文件保存到 `saifu/backend/data/exports/` 目录
4. 返回文件路径和下载 URL

### 2. Word 文档排版要求

文档结构（按顺序）：

---

**封面/标题**
- 居中大标题：「赛赋 SaiFu · 竞赛调研报告」
- 副标题：「为 {姓名} 定制」
- 日期：生成日期
- 下方小字：「由 赛赋 SaiFu 智能生成 · saifu.asia」

**第一节：用户画像速览**
- 表格形式，2 列（项目 | 内容）
- 包含：姓名、学校、专业、年级、兴趣领域、核心技能、参赛目标、时间投入
- 简洁即可，不用全量字段

**第二节：推荐竞赛**
- 每个竞赛一个小标题（如「🏆 中国大学生计算机设计大赛」）
- 下面一个表格：

| 项目 | 内容 |
|------|------|
| 级别 | 国家级 |
| 报名截止 | 2026-06-15 |
| 匹配理由 | 匹配你的 AI/大模型方向... |
| 需要准备 | 准备一个 AI 项目作品... |

- 推荐 5-8 个竞赛

**第三节：备赛建议**
- 正文段落，分点列出
- 包含：时间规划、技能补强建议、组队策略

**第四节：风险提示**
- 如果用户画像有短板（如无指导老师、技能空白），用 ⚠️ 标记
- 给出替代方案

**页脚**
- 每页底部：「赛赋 SaiFu · 智能竞赛匹配 · 微信 g2l0u0e7oi」

### 3. 前端：加导出按钮

在调研结果展示区加一个「📥 导出 Word」按钮：
- 点击后调用 `POST /api/export-report`
- 返回文件路径后，触发浏览器下载
- 下载文件名：`赛赋-竞赛调研报告-{姓名}.docx`

### 4. 安装依赖

需要在 `saifu/backend/requirements.txt` 中添加：
```
python-docx>=1.1.0
```

---

## 约束

- Word 文档必须中文友好（字体用「微软雅黑」或系统默认中文字体）
- 表格要有边框，样式整洁
- 移动端下载也能正常触发
- 如果 `python-docx` 安装有困难，备选方案是用 HTML 生成 → 浏览器打印为 PDF
