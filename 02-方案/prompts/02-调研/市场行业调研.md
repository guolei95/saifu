# 市场/行业调研提示词

## 用途

系统自动调度搜索工具，搜集行业数据、市场规模、政策文件、竞品信息，汇总清洗后输出结构化调研素材。

## 提示词模板

```
# 任务
我正在准备一个大学生竞赛项目，需要你帮我做深度调研。通过提问了解我的情况后，输出一份结构化的调研素材。

# 第一步：信息收集（发送后立刻执行，一次性问完以下问题）

1.「你在准备什么比赛、调研什么方向？」
   示例：「互联网+，方向是校园心理健康AI解决方案，想了解这个领域的市场现状和现有产品」

2.「你现在项目做到什么阶段了？」
   示例：「刚确定方向，还没开始写方案，想先摸清楚行业情况」或「方案初稿写了一半，发现数据不够，需要补充市场规模和竞品信息」
   我会根据你的阶段自动判断应该侧重搜什么（行业现状/竞品/技术可行性/政策数据）。

# 第二步：执行调研并输出结果
按以下结构输出，每个板块独立成段：

1. **关键事实**（5-8条）：这个方向最重要的背景信息，每条附数据来源
2. **核心矛盾**（2-3个）：这个领域最突出的问题是什么，为什么现有方案没解决
3. **值得深挖的线索**（3-5条）：你发现的、大多数人不知道的信息，每条说明为什么值得深挖
4. **数据清单**：列出你引用的所有数据，标注来源和年份。无法确认来源的标注「⚠️ 未验证」

# 输出格式示例
以下是「核心矛盾」的写法示例：

**核心矛盾1：需求爆发但供给专业度不足**
2024年中国在线心理咨询市场规模达XX亿元（来源：艾瑞咨询2024），但持证心理咨询师仅XX万人（来源：中国心理学会）。大量非专业「情感陪聊」平台涌入市场，用户投诉率高达XX%。矛盾在于：用户愿意付费，但找不到靠谱的服务。

# 约束
不要输出超过1500字。我需要的是素材，不是完整报告
不要给建议或方案。只给事实和发现，方案我自己来
中文搜索和英文搜索都要覆盖，海外数据往往更权威
如果你对某个数据没有把握，宁可不写也不要编
```
