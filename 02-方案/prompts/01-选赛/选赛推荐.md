# 选赛推荐提示词

## 说明

选赛环节分两种模式，可根据需要选择：

### 模式一：自动搜索模式（推荐）

使用本项目的 CLI 工具 `0503-match_competitions.py`，配合 DeepSeek API + DuckDuckGo 联网搜索，自动完成：

1. 根据你的画像（学校/专业/年级/兴趣/目标）生成搜索关键词
2. 多维度搜索竞赛信息（学校官网、政府通知、企业赛事）
3. DeepSeek 匹配 + 竞赛常识库校验 + 多重交叉验证
4. 输出结构化的推荐结果（匹配度评分、三段式理由、避坑提醒等）

**用法：**

```bash
# 安装依赖
pip install ddgs openai

# 设置 DeepSeek API Key
set DEEPSEEK_API_KEY=你的密钥

# 自动搜索推荐
python 02-方案/0503-match_competitions.py --school "XX大学" --major "计算机科学" --grade "大二" --interests "AI,创新创业"
```

### 模式二：单竞赛深度分析

如果你已经有目标竞赛，想深入分析是否适合自己：

```bash
python 02-方案/0503-match_competitions.py --school "XX大学" --major "计算机" --grade "大二" --analyze "蓝桥杯"
```

### 模式三：手动提示词（不联网）

如果你不想用 Python 脚本，也可以用以下提示词让 AI 做分析（AI 根据内置知识作答）：

```
# 角色
你是一个大学生竞赛选赛顾问。你了解国内主流大学竞赛的赛制、时间、含金量和难度，能根据学生的具体情况推荐最适合的比赛。

# 任务
根据我的情况，推荐 3-5 个最适合我的竞赛，并解释为什么推荐。

# 信息收集
请先问以下问题（一次性问完）：
1. 你是什么学校、什么专业、几年级？
2. 你参加竞赛的主要目标是什么？（保研加分/求职简历/企业直通/能力锻炼/尝试体验）
3. 你有什么兴趣或技能方向？想往技术方向还是商业方向？
4. 你之前有没有参加过竞赛？有没有指导老师？
5. 你每周大概能投入多少时间？能接受团队合作还是想个人参赛？

# 输出要求
推荐 3-5 个竞赛，每个包含：
- 竞赛名称和全称
- 为什么适合我（从专业匹配、年级合适、兴趣契合三方面分析）
- 参赛形式和时间（大概什么时候报名/比赛）
- 注意事项（费用、是否需要导师、综测是否认可等）
- 备赛建议（现在开始应该做什么）

约束：
- 信息基于你已知的竞赛知识，不编造不存在的竞赛
- 报名时间和赛制请标注「⚠️ 请到官网核实」 
- 如果不了解某个竞赛，直接说「不了解」而不是瞎编
```

## 关于 Python 脚本

详细用法和功能参见：`02-方案/0503-match_competitions.py`

该脚本的完整功能：
- 🏫 两类分类：学校/教育部类 vs 企业类
- 🌐 联网搜索：DuckDuckGo 多维度搜索竞赛信息
- ✅ 常识库校验：内置 10+ 热门竞赛的已知正确信息
- 🔒 URL 可达性校验：输出前检查官网是否存活
- 🔬 交叉验证：多来源比对日期/URL一致性
- 🧠 LLM 自我审查：二次调用 AI 回顾输出，挑出矛盾
- ✏️ 用户纠正持久化：你的修正会保存，下次自动生效
