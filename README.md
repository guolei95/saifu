# 竞赛项目模板 - 使用说明

这是一个 AI 自动管理的竞赛项目文件夹。解压后用 Cursor 打开，AI 会自动帮你管理项目进度、整理文件、记录日志。

---

## 快速上手（3步）

### 第一步：解压

- **Mac**：双击 zip 文件即可
- **Windows**：右键 → 解压到当前文件夹
  - 如果解压后文件名显示乱码，下载 [7-Zip](https://www.7-zip.org/) 重新解压

### 第二步：用 Cursor 打开

打开 Cursor → 文件 → 打开文件夹 → 选择「我的竞赛项目」文件夹

> 为什么要用 Cursor 打开？因为 Cursor 的 AI 能自动读取文件夹里的规则文件（CLAUDE.md），知道怎么帮你管项目。直接用网页版 AI 聊天做不到这一点。

### 第三步：和 AI 对话

在 Cursor 里按 `Ctrl+L`（Mac 按 `Cmd+L`）打开 AI 对话，随便说一句话就行。

AI 会自动检测到这是新项目，问你几个问题（参加什么比赛、几个人、做到哪了），然后自动帮你配置好一切。

---

## AI 会自动帮你做的事

- **管文件**：写的每个文件自动放到正确的文件夹，不用你操心
- **记进度**：每完成一件事自动在进度表打勾
- **写日志**：关键决策和完成节点自动记录
- **接上次**：每次打开新对话，AI 自动告诉你上次做到哪了

## 你只需要做的事

1. 回答 AI 的问题
2. 把手上的素材（往届作品、竞赛通知等）丢给 AI
3. 看 AI 的产出，说「行」或「这里要改」
4. 自己去提交比赛（这个 AI 做不了）

## 如果你不用 Cursor

如果你用的是网页版 AI（ChatGPT / 豆包 / Kimi 等）：

1. 打开 `CLAUDE.md` 文件，复制全部内容
2. 在网页版 AI 的对话开头粘贴，告诉 AI：「以下是我的项目规则，请按这个规则帮我工作」
3. 每次新对话都需要重新粘贴（网页版 AI 没有跨对话记忆）
4. 文件管理需要你自己手动操作（网页版 AI 无法直接操作你电脑上的文件）

> 建议还是用 Cursor，体验好很多。
