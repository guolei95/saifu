# 竞赛选赛助手 (Competition Match)

根据用户画像，联网搜索当前可报名的竞赛，智能匹配并输出推荐结果。

## 使用方式

三种方式任选：

1. **一句话调用**（最简单）：
   ```
   /competition-match
   ```
   AI 会自动读取 `03-素材/用户画像模板.md` 获取你的信息，然后搜索+匹配。

2. **手动分析单个竞赛**：
   ```
   /competition-match --analyze "蓝桥杯"
   ```

3. **直接双击** `运行.bat`（不需要 AI）。

## 运行原理

```
用户画像 → DuckDuckGo搜索(30+关键词) → DeepSeek匹配 → 
L3常识库校验 → L1交叉验证 → L2 LLM自审查 → 格式化输出
```

输出内容：竞赛名、匹配度评分、三段式理由（专业/年级/兴趣）、参加好处、避坑提醒、推荐指数、截止日期、费用、官网。

校验标识：
- 📌 = 已用常识库校验
- 🟢/🔴 = 交叉验证置信度
- 🧠 = AI自我审查发现的问题

## 前置条件

- Python 3.x + `pip install ddgs openai`
- DeepSeek API Key（环境变量 `DEEPSEEK_API_KEY` 或首次运行时输入）
- 用户画像模板已填写（`03-素材/用户画像模板.md`）

主脚本：`02-方案/0503-match_competitions.py`
画像模板：`03-素材/用户画像模板.md`
一键启动：`运行.bat`
