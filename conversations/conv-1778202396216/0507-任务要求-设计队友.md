### 配图工作节奏

```
各章陆续定稿后你来画图（按定稿顺序）：

早期: 第4章定稿（小雷）→ 产品架构图 + 用户流程图
早期: 第3章定稿（商业A）→ 市场规模可视化
中期: 第5章定稿（小雷）→ 技术架构图
中期: 第6章定稿（商业B）→ 商业模式画布
中期: 第7章定稿（商业B）→ 竞品对比表 + 获客路径图
后期: 第8章定稿（小雷）→ 成长时间线图
后期: 第9章定稿（商业A）→ 社会价值示意图
后期: 第10章定稿（商业A）→ 甘特图
最后: 第1、12章定稿（小雷）→ 封面 + 照片排版
```

### 配图注意事项

- 各章负责人给你的文字要**已定稿**的版本，半成品不接
- 如果文字有不清楚的地方，直接找对应负责人确认（别自己猜）
- 统一视觉风格：所有图用同一套配色方案（建议：品牌蓝 #2563EB + 渐变）

## 第二部分：独立产出（可入作品集）

### 附录B：产品原型与界面展示 ⭐⭐⭐

**交付物**：

- 3-5张核心页面原型（Figma高保真）
- 500字设计说明

**建议覆盖的页面**：

| 页面 | 展示什么 |
| --- | --- |
| 首页/登录页 | 双路径入口：「我知道要参加什么」vs「我不知道」 |
| 选赛推荐页 | 输入专业年级 → AI推荐比赛列表 |
| 方案工作台 | 方向调研 → 框架生成 → 逐节写作的工作流界面 |
| 知识库浏览 | 300+赛事库 + 方案模板库的浏览界面 |
| 个人中心 | 方案存档 + 竞赛日历 + 赛后沉淀 |

**参考素材**：

- `04-PPT/成品PPT/赛赋-项目展示.html` — 已有界面示意方向
- `02-方案/0506-方案框架-赛赋.md` — 功能设计详细描述
- `02-方案/prompts/` — 功能背后的提示词逻辑（帮助理解交互）

**做原型时注意**：

- 重点是MVP版本的网页端（微信小程序是后续规划）
- 配色参考已有的策划文档风格
- 每个页面加一句设计说明：这页解决了用户什么需求
