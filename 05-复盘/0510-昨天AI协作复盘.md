# 0510 昨天AI协作复盘

## 做了什么

同时开了三个聊天窗口，分别让牛马AI + DeepSeek 并行改代码：
1. **搜索引擎** —— 一个聊天专门干搜索功能
2. **内置信息库** —— 一个聊天改知识库
3. **手机端网站** —— 一个聊天改移动端适配

## 核心感受

- **非代码技术也能独立开发**：不需要会写代码，只需要把问题说清楚，AI就一直改到对为止
- **并行效率爆炸**：三个聊天同时推进，相当于三个人同时开发，但只需要一个人描述需求
- **DeepSeek性价比逆天**：十几个小时的开发，花了不到10块钱。如果用 Claude Code 同样的事可能要五六十块

## 成本对比

| 工具 | 预估费用 | 备注 |
|------|---------|------|
| DeepSeek | <¥10 | 国产，性价比之王 |
| Claude Code | ¥50-60 | 更强但更贵 |

## 心得

> 真正意义上实现了非代码技术人员独立开发网站、开发应用的程度。AI写代码的能力简直逆天。
