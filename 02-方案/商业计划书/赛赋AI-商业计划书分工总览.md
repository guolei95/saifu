## 为什么要拆成 2 个商业队友

```
队友A：需求 → 市场 → 社会价值（证明"值得做、有何社会意义"）
队友B：模式 → 营销 → 落地 → 财务 → 团队 → 风险（证明"能赚钱、能持续、有人做"）

两组各自有完整逻辑链，互不阻塞。Ch10从A转B是因为B写完模式/营销后对商业化路径更熟。
```
