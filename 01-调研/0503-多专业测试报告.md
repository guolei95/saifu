## 🔑 关键发现

### 1. 费用情况（DeepSeek 大幅改善）

| 竞赛          | 费用         | 状态          |
| ----------- | ---------- | ----------- |
| 蓝桥杯         | 300元/人     | ✅ 正确标注      |
| 语言文字能力大赛    | 49元/人      | ✅ 正确标注      |
| 汉语言文字能力大赛   | 50元/人      | ✅ 正确标注      |
| 商业精英挑战赛(会计) | 30元/人(知识赛) | ✅ 原文提到但未匹配到 |
| NECCS       | 50元/人      | ✅ 正确标注      |
| 大部分学科竞赛     | 免费         | ✅ 正确标注      |

### 2. 截止日期问题（依然存在）
- 约 60% 竞赛截止日期标注为"未知"
- 原因：竞赛官网往往不在一篇通知里写清全部时间线
- 改进方向：加「预计报名时间」字段，用往年规律推测

### 3. 专业差异

| 专业类型 | 春季(3-6月)可报  | 秋季(9-11月)可报 | 特点         |
| ---- | ----------- | ----------- | ---------- |
| 文史类  | ✅ 写作/语言文字赛多 | 较少          | 个人赛为主，门槛低  |
| 经管类  | 较少          | ✅ 财会/金融赛集中  | 团队赛为主，周期长  |
| 理工类  | 部分（力学/BIM）  | ✅ 结构/计算机赛   | 动手实践，准备周期长 |

### 4. 大一适配度
- 🟢 高适配：写作类、语言文字类、英语竞赛、力学竞赛（个人参赛）
- 🟡 中适配：创新创业类、调研类（可做基础工作）
- 🔴 低适配：模拟法庭、结构设计（需要较多专业知识）
